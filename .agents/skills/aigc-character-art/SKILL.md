# AIGC 形象美术设计 Skill

在用户要求你为角色、场景或核心视觉形象设计 AIGC 美术方向时使用本 skill。

## 适用场景

- 用户要你设计角色形象、人设视觉、服装、材质、气质与辨识度
- 用户要你形成可用于图像生成或角色卡片的视觉设定
- 用户要你把剧情、角色功能与视觉风格统一起来

## 工作方法

1. 先确认角色在剧情中的定位与叙事功能，不要把美术设计与角色功能割裂。
2. 设计时优先明确：
   - 核心身份与气质关键词
   - 轮廓、体态、服装层级、材质与配色
   - 年龄感、时代感、职业感、世界观一致性
   - 与其他角色的差异化点
3. 如果项目里已有角色档案、风格指南或世界观约束，先读取再设计。
4. 输出应尽量能直接转成角色卡、视觉 brief 或图像提示词素材。

## 推荐输出框架

- 视觉定位
- 造型关键词
- 服装 / 材质 / 配色
- 表情与姿态倾向
- 可直接用于生成的描述要点

## 约束

- 不要只给抽象形容词，不给可执行的视觉特征。
- 不要忽视项目已有风格约束。
- 不要把角色设计做成与剧情功能无关的空洞“美型设定”。
