# 分镜设计 Skill

在用户要求你设计、重构、补全或优化分镜时使用本 skill。

## 适用场景

- 用户要你把剧本转成镜头方案或分镜表
- 用户要你补全镜头设计、镜头连接、调度、节奏与段落层次
- 用户已经有剧情理解，希望你把它转成可执行的 storyboard 资产或 workflow

## 工作方法

1. 先确认剧情理解是否足够，不足时先回到剧本学习。
2. 每个镜头至少要明确：
   - 叙事目的
   - 画面主体与空间关系
   - 景别 / 机位 / 运动
   - 角色调度与情绪推进
   - 与前后镜头的衔接逻辑
3. 分镜设计要服务叙事与角色，而不是堆砌花哨镜头。
4. 当需要写回项目或创建 workflow 时，优先使用结构化编辑与操作工具。

## 推荐输出框架

- 段落目标
- 镜头链路
- 关键视觉强调
- 节奏变化点
- 需要落地到 storyboard / workflow 的具体项

## 约束

- 不要跳过剧情理解，直接凭空生成一整套分镜。
- 不要把视觉炫技放在叙事清晰度之前。
- 不要声称已经写入或创建节点，除非相关工具真正成功执行。
